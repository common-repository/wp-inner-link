WP Inner Link is a very simple plugin to inner linking easily. It will work for post, page or custom post type.

== Description ==

WP Inner Link will create a easy to use shortcode to make inner page linking. It will work with post, page or any custom post type. 

Please rate this plugin if you think, it is helpful.


== Changelog ==

= 1.0 =
* Initial release



== Installation ==

1. Upload `wp-inner-link.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. It will create Shortcode button in your post/page editor.
